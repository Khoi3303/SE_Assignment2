4. HOW TO RUN THE PROJECTS:
- Step 1: Clone or download the repository from Github.
- Step 2: Open the local folder using Visual Studio 2022.
- Step 3: Right-click on the Solution -> select "Restore NuGet Packages" to download all required libraries.
- Step 4: Database Setup:
   + For Exer 1, 2, 3: Use the provided SQL script to generate the "SE_Assignment2_DB" database in your LocalDB.
   + For Exer 4 (Code First): Open Package Manager Console, select project Exer 4, and run command: "Update-Database" to automatically generate the database.
- Step 5: Press Ctrl + F5 to build and run each project.

Note: The Unit Test project (Exer 5) requires running from the Test Explorer in Visual Studio.
